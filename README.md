## Features

| الميزة | الوصف |
|--------|-------|
| **دمج PDF** | دمج عدة ملفات PDF في ملف واحد |
| **فك الدمج** | استخراج صفحات محددة من ملف PDF |
| **إزالة كلمة المرور** | إزالة كلمة المرور من ملف PDF محمي |
| **PDF إلى صور** | تحويل صفحات PDF إلى صور |
| **صور إلى PDF** | إنشاء PDF من صور |
| **دمج الصور** | دمج عدة صور في صورة واحدة |

## How to Use

### نسخة الويب (PDF-Toolbox.html)
افتح ملف `PDF-Toolbox.html` مباشرة في المتصفح

### نسخة PWA
افتح المجلد `pdf-toolbox-pwa/` وشغّل `index.html`
